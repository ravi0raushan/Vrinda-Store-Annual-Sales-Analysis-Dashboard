

# 📊 Vrinda Store Annual Sales Analysis Dashboard

## Project Overview

This project analyzes Vrinda Store sales data using Microsoft Excel to uncover sales trends, customer behavior, order patterns, and channel performance. The analysis was performed using data cleaning, pivot tables, pivot charts, slicers, and an interactive dashboard.

The goal was to answer key business questions such as:

* Which customer segment contributes the most revenue?
* Which sales channels perform best?
* Which states generate maximum sales?
* What is the relationship between orders and revenue?
* Which age and gender groups place the most orders?

---

## Dataset Features

The dataset includes:

* Order ID
* Customer ID
* Gender
* Age and Age Group
* Order Date and Month
* Order Status
* Sales Channel (Amazon, Myntra, Ajio, Flipkart, Meesho, Others)
* Product Category
* Product Size
* Quantity Ordered
* Sales Amount
* Shipping City and State

## Dashboard Insights

### Key Findings:

### 1. Women customers contribute more sales than men

* Female customers are the dominant revenue contributors.
* Indicates stronger purchase behavior among women.

### 2. Adult age group contributes maximum orders

* Adult customers are the most valuable customer segment.
* Primary target audience for marketing campaigns.

### 3. Top 5 states generate majority of sales

* High revenue concentration from top-performing states.
* Useful for geo-targeted advertising and inventory planning.

### 4. Sales are driven heavily by online marketplaces

* Channels like Amazon, Myntra, and Ajio contribute major share.
* Marketplace optimization can increase revenue further.

### 5. Delivered orders dominate order status

* Healthy fulfillment performance.
* Lower cancellations/returns improve profitability.

### 6. Orders and sales trend show seasonal patterns

* Useful for forecasting demand and planning promotions.
